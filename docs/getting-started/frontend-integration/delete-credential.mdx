# 3. Delete a Credential

When a user wants to delete a collector/credential from your app, he can click on the Delete button in the table.

Flow:
1. User click on the `Delete` button in the table
2. Delete credential with endpoint `DELETE /user/{user_id}/credential/{credential_id}`
3. Update credential from table

```mermaid
sequenceDiagram
    participant User
    participant Frontend as Your Frontend
    participant Backend as Your Backend
    participant API as Invoice-Collector API

    User->>Frontend: Clicks "Delete" button
    Frontend->>Backend: Request credential deletion
    Backend->>API: DELETE /user/{user_id}/credential/{credential_id}
    API-->>Backend: Returns
    Backend-->>Frontend: Render new table
    Frontend-->>User: Display collectors list
```
